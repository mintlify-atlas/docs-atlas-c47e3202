This guide walks you through running the **Credit Scoring** service — a credit risk prediction API backed by a PyTorch MLP model. By the end you will have a working REST endpoint on your machine.

<Note>
  All commands run from the repository root (`deep_learning_services/`). Make sure you are in that directory before executing any `docker build` or `docker run` commands.
</Note>

## Prerequisites

Before you begin, ensure the following tools are installed:

| Tool | Minimum version | Install guide |
|------|----------------|---------------|
| Docker | 24.x | [docs.docker.com](https://docs.docker.com/get-docker/) |
| Git | 2.x | [git-scm.com](https://git-scm.com/downloads) |

## Run the Credit Scoring service

<Steps>

<Step title="Clone the repository">

```bash
git clone https://github.com/AprendeIngenia/deep_learning_services.git
cd deep_learning_services
```

</Step>

<Step title="Build the Docker image">

Run the following command from the repository root. The build uses the optimized multi-stage Dockerfile in `container-images/credit_scoring_service/`.

```bash
docker build -t ingeniia/credit-scoring-mlp:1.0 -f container-images/credit_scoring_service/Dockerfile .
```

<Tip>
  The first build takes a few minutes while Docker pulls the base image and installs Python dependencies. Subsequent builds are faster thanks to layer caching.
</Tip>

</Step>

<Step title="Run the container">

Start the service in detached mode and bind it to port `8000` on your host:

```bash
docker run -d -p 8000:8080 --name credit-scoring-service ingeniia/credit-scoring-mlp:1.0
```

Confirm the container is healthy by tailing its logs:

```bash
docker logs -f credit-scoring-service
```

You should see Uvicorn report that the application started successfully.

</Step>

<Step title="Explore the interactive docs">

Open your browser and navigate to:

```
http://localhost:8000/docs
```

FastAPI automatically generates a Swagger UI where you can inspect the `/mlp_demo` endpoint and try it out interactively without writing any code.

</Step>

<Step title="Make your first prediction">

Send a POST request to `/mlp_demo` with an applicant's financial profile:

```bash
curl -X 'POST' \
  'http://localhost:8000/mlp_demo' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "Age": 35,
  "Sex": "male",
  "Job": 2,
  "Housing": "own",
  "Saving accounts": "little",
  "Checking account": "moderate",
  "Credit amount": 2500,
  "Duration": 24,
  "Purpose": "car"
}'
```

</Step>

<Step title="Verify the response">

A successful prediction returns HTTP `200 OK` with the following JSON body:

```json
{
  "prediction": "good",
  "probability": 0.7852
}
```

- **`prediction`** — `"good"` or `"bad"` credit risk classification produced by the MLP model.
- **`probability`** — confidence score that the applicant falls into the `"good"` category (range: 0 – 1).

</Step>

</Steps>

## Manage the container

A few useful commands for the container lifecycle:

```bash
# Stream logs in real time
docker logs -f credit-scoring-service

# Stop the service
docker stop credit-scoring-service

# Restart a stopped container
docker start credit-scoring-service

# Remove the container entirely
docker stop credit-scoring-service && docker rm credit-scoring-service
```
