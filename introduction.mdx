Deep Learning Services is a collection of three production-grade AI microservices built as part of the [inGeniia.co](https://www.ingeniia.co) Deep Learning course. Each service wraps a trained neural network in a FastAPI application, containerized with Docker and deployable to Google Cloud Run.

This is not a collection of Jupyter notebooks — it is structured, MLOps-grade code designed to run in the cloud.

## Key capabilities

- **FastAPI** — High-performance REST and WebSocket endpoints with auto-generated Swagger docs
- **Pydantic validation** — Strict type-safe request/response schemas for all services
- **Docker** — Multi-stage, optimized container images ready for cloud deployment
- **Google Cloud Run** — Serverless, auto-scaling deployment with CI/CD via Cloud Build
- **GradCAM explainability** — Visual heatmaps showing model attention regions for X-ray classification
- **GPU inference** — WebSocket-based real-time detection with CUDA support in the Smart Checkout service

## Project structure

```text
deep_learning_services/
├── container-images/       # Optimized Dockerfiles for each service
├── ops/                    # Cloud Build CI/CD manifests for GCP
├── python/
│   ├── credit_scoring/     # MLP credit risk prediction service
│   ├── xrays_evaluation/   # CNN chest X-ray classification service
│   └── smart_checkout/     # YOLO real-time object detection service
└── cloudbuild.yaml         # Root-level Cloud Build trigger
```

<Note>
  All three services are independent microservices. You can deploy any one of them without the others.
</Note>
